# Assets

## Screenshot Instructions

To add a preview screenshot:

1. Open `index.html` in your browser
2. Take a screenshot of the full interface (graph + sidebar)
3. Save as `preview.png` in this folder
4. Recommended size: 1920x1080 or 2560x1440

## Optional Additional Assets

You can add:
- `favicon.ico` - Browser tab icon
- `og-image.png` - Social media preview (1200x630)
- `demo.gif` - Animated demo of interaction

# 🌐 GitHub Pages Setup Guide

## Quick Setup (5 minutes)

Follow these steps to make your knowledge graph accessible online!

---

## 📋 Step-by-Step Instructions

### Step 1: Go to Your Repository Settings

1. Open your browser and go to:
   ```
   https://github.com/Robertren/ai-agent-interview-mind-graph
   ```

2. Click the **"Settings"** tab at the top
   - It's the last tab in the menu (after Insights)
   - ⚙️ Gear icon

---

### Step 2: Find GitHub Pages Settings

1. In the left sidebar, scroll down to find **"Pages"**
   - It's under the "Code and automation" section
   - Click on **"Pages"**

---

### Step 3: Configure Source

On the Pages settings page:

1. **Under "Source"**, you'll see a dropdown that says "None"

2. Click the **dropdown** and select:
   - Branch: **`main`**
   - Folder: **`/ (root)`**

3. Click the **"Save"** button

---

### Step 4: Wait for Deployment

1. After clicking Save, you'll see a message:
   ```
   Your site is ready to be published at
   https://robertren.github.io/ai-agent-interview-mind-graph/
   ```

2. **Wait 1-2 minutes** for the initial deployment
   - GitHub needs to build and deploy your site
   - You can refresh the page to check status

3. When ready, you'll see:
   ```
   ✅ Your site is published at
   https://robertren.github.io/ai-agent-interview-mind-graph/
   ```

---

### Step 5: Visit Your Live Site!

Open this URL in your browser:
```
https://robertren.github.io/ai-agent-interview-mind-graph/
```

🎉 **Your knowledge graph is now live and accessible to anyone!**

## 🔄 Updating Your Site

### Every Time You Push to GitHub:

1. Make changes locally
2. Commit: `git commit -m "Update content"`
3. Push: `git push`
4. **GitHub Pages auto-updates in 1-2 minutes!**

No need to do anything else - it's automatic!

## 🐛 Troubleshooting

### Issue: "404 - Page Not Found"

**Solution:**
1. Check that `index.html` is in the root folder
2. Wait 2-3 minutes for initial deployment
3. Try clearing browser cache (Ctrl+Shift+R)

### Issue: "Site Not Updating"

**Solution:**
1. Check your latest commit is on `main` branch
2. Go to Actions tab to see deployment status
3. Wait 2-3 minutes for changes to propagate

### Issue: "Files Not Loading"

**Solution:**
1. Check that `data/` folder is in the repository
2. Verify file paths are relative (not absolute)
3. Check browser console (F12) for errors

## 📊 GitHub Actions (Automatic)

GitHub automatically creates a workflow for Pages.

### To View Deployment Status:

1. Go to **"Actions"** tab in your repository
2. See deployment history
3. Each push creates a new deployment

### Deployment Process:
```
1. You push code → GitHub
2. GitHub detects change
3. Builds static site
4. Deploys to Pages (1-2 min)
5. ✅ Site updated!
```
